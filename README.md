# Burnable , Mintable and Pausable ERC20 Smart Contract
# Upgradeable Staking contract 
# V1 - static
# V2 - dynamic

## Installing dependencies

```
npm install
```

## Testing the contract

```
npm test
```

## Deploying the contract

```
npx hardhat run --network <network-name> scripts/deploy.ts
```
